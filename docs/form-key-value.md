# form-key-value

form-key-value组件为键值对组件。

---

| 属性    | 说明 | 类型 |默认值|
| ------ | ------ | ------ |------ |
| isNewCom  | 是否带有表头的键值对 | Boolean |false|
  

> isNewCom  

当isNewCom属性为true时，显示有表头. 
![avatar](/img/form-key-value/isNewCom.png)
<br/>
当isNewCom属性为false时，显示无表头.
![avatar](/img/form-key-value/isNewCom-false.png)
